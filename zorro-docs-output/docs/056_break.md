# break

## break

The **break** statement terminates a **while** or **for** loop or a **switch..case** statement, and continues with the first instruction after the closing bracket.

### Example:

```c
while (x < 100)
{ 
  x+=1;
  if (x == 50) break; _// loop will be ended prematurely_
}
```
