# sqrt

## sqrt(var x): var

Square root of x.

### Parameters:

**x** - any **var**.  

### Returns:

Square root of x.

### Example:

```c
x = sqrt(2); _// x is now 1.414_
x = sqrt(64); _// x is now 8_ 
x = sqrt(1); _// x is now 1_
```
